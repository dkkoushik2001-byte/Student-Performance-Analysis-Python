# Student Performance Analysis

This project analyzes student performance data using Python. The objective is to identify factors that influence student grades and gain insights through data cleaning, exploratory data analysis, and visualization.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- School-wise Performance Analysis
- Gender-wise Performance Analysis
- Age-wise Analysis
- Pass/Fail Analysis
- Top and Low Performing Students
- Data Visualization

## Key Insights
- Compared student performance across schools.
- Analyzed grade distributions and pass rates.
- Identified top-performing students.
- Explored the relationship between study factors and final grades.
